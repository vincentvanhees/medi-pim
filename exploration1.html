test = zoo::rollapply(D[, c("MEDI", "PIM")],
                 width = n_epoch_per_minute * 5, by = n_epoch_per_minute * 5, function(x) ifelse(sd(x[,1]) != 0 && sd(x[,2]) != 0, cor(x[, 1], x[, 2], use = "complete.obs"), 0), by.column = FALSE)
D$corr = c(rep(test, each = n_epoch_per_minute * 5), rep(0, 15))

myplot(D |> filter(corr > 0.6))

Downsample PIM and MEDI to 1 minute resolution?

D$rounded_time_minutes = round(as.numeric(D$Datetime) / 60) * 60

myagg = function(x) {
  if (inherits(x, "numeric")) {
    return(mean(x))
  } else {
    return(x[1])
  }
}
Dtmp <- as_tibble(aggregate(D, by = list(D$rounded_time_minutes, D$Id), FUN = myagg)) |>
  select(-c(Group.1, Group.2))

Reflections

  • No clear associations between PIM and MEDI or change in PIM and change in MEDI.
  • No clear association when focussing only on day or night, summer or winter.
  • No clear association after changing the resolution in values or in time.
  • Possible reasons:
    • All data per site combined, zooming in on a smaller subset or even individual recordings may provide more insight.
    • Only looked at isolated effect of each factor, not at the combined effect.
    • Limited attempt to clean data
    • Only focused on epoch level comparisons
